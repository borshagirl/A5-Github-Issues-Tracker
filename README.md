
***question No-1: What is the difference between var, let, and const?

Answer No-1: var--> 1.Function scoped, 2.allowed Re-declare and 3.allowed Re-assign.
  let--> 1.It's a blocked scoped, 2.don't allow Re-declare and 3.allowed Re-assign.
  const--> 1.it's a block scoped, 2.don't allow Re-declare and Re-assign.


***question No-2:  What is the spread operator (...)?

Answer No-2: Spread operator (...) is used to expand elements of an array or object into individual elements.


***question No-3: What is the difference between map(), filter(), and forEach()?

Answer No-3: map() --> Creates a new array by changing each element.

   filter() --> Creates a new array with elements that pass a condition.

   forEach() --> Loops through the array but does not return a new array.
